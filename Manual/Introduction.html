Introduction

Extraction piste son est un logiciel gratuit réalisé par un groupe d'étudiant possèdant une interface pour lire, créer, modifier des fichier Midi. Le programme est capable d'ouvrir des fichiers Midi existant et de modifier leur contenu. Une fonctionnalité est disponible afin de créer des partition musicales a partir de fichier Midi et de les modifier.

Ce manuel fournira une vue d'ensemble sur les differentes fonctionnalité de Extraction piste son. Toutefois si nous souhaitons discuter des fondements du Midi, nous allons fournir un aperçu ici.

Note

Extraction piste son a était developpé par un groupe d'etudiant de l'ecole Dhuoda. C'est entièrement écrit en C++ (Qt) et est disponible sur la platforme Windows.
