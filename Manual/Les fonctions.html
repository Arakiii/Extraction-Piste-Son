Les fonctions

Extraction_piste_son possède plusieurs fonctions permettant un bon déroulement du logiciel.

Dans cette partie vous retrouverai toute les fonctions disponible dans la version 0.5 du logiciel:

 

Ouvrir Fichier Midi

Cette fonction permet de choisir le fichier Midi sur lequel le logiciel va travailler permettant d'utilisé les autres fonctions du logiciel.

L'opération de cette fonction vous demande d'indiqué l'endroit ou le fichier Midi se trouve.

 

Chemin d'accès

Cette fonction permet d'afficher l'endroit ou se situe l'objet sur lequel le logiciel est en train de travailler.

L'opération de cette fonction se réalisé dés que l'utilisateur rentre en contact avec la fonction "Ouvrir Fichier Midi" autrement il est impossible d'intéragir avec.

 

Création Partition

Cette fonction permet d'effectuer la création de la partition en examinant le fichier Midi le logiciel compare les données du fichier Midi a un tableau interne afin de pouvoir réalisé la partition.

L'opération de cette fonction demande d'indiqué l'endroit d'enregistrement.

 

Ouvrir Partition

Cette fonction permet d'ouvrir dans un format plus élevée que la prévisualisation une image de la partition du choix de l'utilisateur.

L'opération de cette fonction demande d'indiqué l'emplacement ou se trouve l'image.

 

Modifier Partition

Cette fonction permet d'ouvrir l'editeur de partition afin de pouvoir effectuer divers opération sur la partition ( En travaux ).

L'opération de cette fonction demande un fichier Midi de sélectionner au préalable.

 

Enregistrer PDF

Cette fonction permet d'enregistrer sa partition dans un format lisible par tout les système informatique existant.

L'opération de cette fonction demande d'indiqué l'emplacement ou se trouve l'image que l'on souhaite transformé en PDF et l'emplacement de l'enregistrement.

 

Imprimer

Cette fonction permet d'imprimer sa partition en format papier a travers une imprimante.

L'opération de cette fonction demande d'indiqué l'imprimante avec laquelle le logiciel doit communiquer les information.

 

Prévisualisation

Cette fonction permet d'afficher une prévisualisation de la première partition qui vient d'etre créer.

Une fonction secondaire permet d'agrandir l'image de la prévisualisation à l'aide d'un simple clique sur la fenetre de prévisualisation.

L'opération de cette fonction se réalise dès que l'utilisateur rentre en contact avec la fonction "Création Partition". L'opération de la fonction secondaire peux se réalisé a n'importe quelle moment a l'aide d'un simple clique sur la fenetre.

 

Lecteur Midi

Cette fonction permet d'ecouter le fichier Midi est d'intéragir avec la musique.

Le bouton de gauche permet de mettre en pause la musique

Le bouton de droite permet de reprendre la musique ou elle était arreter.

L'onglet "Speed" Permet de régler la vitesse de la musique allant de 0.10x jusqu'a 1.00x.

Le slider permet de ce déplacer dans la timeline (frise chronologique) de la musique.

L'opération de cette fonction se réalise dès que l'utilsateur rentre en contact avec la fonction "Ouvrir Fichier Midi", le lecteur est inaccessible autrement.

 

Piano Roll

Cette fonction permet d'afficher le contenu d'un fichier Midi dans une forme autre que la partition.

L'opération de cette fonction se réalise dès que l'utilisateur rentre en contact avec la fonction "Création partition", Le piano Roll reste vide tant que l'intéraction avec la fonction n'a pas lieu.
