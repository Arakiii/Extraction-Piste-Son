Methode d'utilisation

Extraction_piste_son à était programmé pour fonctionner dans un ordre précis ici ce trouve la methode d'utilisation vous permettant de faire fonctionner le logiciel de A à Z simplement en suivant les étapes qui vous seront proposé ci-dessous

 

Fichier Midi

En effet le logiciel tourne autour de ce fichier. Dans un premier temps l'utilisateur dois introduire son fichier Midi afin de pouvoir continuer d'évoluer dans le logiciel.

Pour cela il suffit de sélectionner le bouton "Ouvrir Fichier MIDI"

Une fenetre de dialogue vous demandera de sélectionner le fichier Midi avec lequel vous souhaitez travailler.

Une fois le fichier sélectionner veuillez vous assurez que le chemin d'accès est bien mis à jour comme ceci :

Si ce n'est pas le cas veuillez recommencer le processus précedent.

 

Création Partition

Une fois l'étape précédente effectuer il vous manque plus que créer la partition pour cela vous remarquerez que au centre de l'écran se trouve plusieurs bouton à votre disposition.

Pour votre cas il suffit de sélectionner le bouton "Création partition"

Une fenetre de dialogue vous demandera de choisir l'emplacement ou vous souhaitez que les partitions s'enregistre.

Une fois cela un aperçu de la partition sera disponible dans la fenetre de prévisualisation et une fenetre de dialogue vous demandant si la partition vous semble correcte ou non (Il existe une possibilité que la partition ne sois pas parfaite il ne faut pas hésiter a dire non).

En sélectionnant "Oui" vous aurez de nouveau accès a tout les autres fonctionnalité du logiciel 

En sélectionnant "Non" le logiciel vous déplacera automatiquement sur la fenetre de modification manuel afin de vous donner la possibilité de modifier la partition comme vous le souhaitez.

 

Finalisation

Maintenant que les modifications sur les partition sont terminé il existe deux fonction que vous pouvez utilisé afin de finir la boucle

La première concerne l'enregistrement en PDF

Pour cela il suffit de sélectionner le bouton "Enregistrement en PDF"

Une fenetre de dialogue vous demandera de sélectionner la partition que vous souhaiter enregistrer en PDF et vous demandera de choisir l'emplacement ou vous souhaitez que la partition sois enregistrer en PDF.

A la fin du dialogue la partition sera transformer en format PDF a l'endroit que vous avez sélectionner.

La deuxième solution concerne l'impression de la partition

Pour cela il suffit de sélectionner le bouton "Imprimer"

Une fenetre vous demandera de sélectionner l'imprimante de votre choix afin que l'impression puisse s'effectuer

(La partition imprimée sera la partition de la prévisualisation)

Une fois l'impression parametrér l'impression s'effecturera.
